How to make a website in HTML

  1. The first step is to open up notepad, visual studio code, or any other code format.
  2. Once you are ready to start writing your code you should start with a DOCTYPE html tag
    • This tag should then be followed by an html tag as well as a head and body tag
  3. After you have completed these steps you will have your HTML website
